# site-mosaico
A Mosaico e um espaco criado para transformar conhecimento em experiencias reais. Localizada em Copacabana, oferecemos uma estrutura profissional pensada para cursos presenciais, workshops, producao de conteudo e experiências gastronômicas. Mais do que salas, a Mosaico e um ambiente preparado para quem ensina, cria e compartilha conhecimento.
